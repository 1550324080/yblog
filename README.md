# springBoot博客系统

#### 介绍
springBoot博客系统，该项目简单上手，非常适合初学者学习，完成毕业设计绰绰有余，该有的功能都有，作者也在
完善当中，希望大家来指点指点，一起学习，共同进步。

#### 技术栈
* **springBoot**
* **rabbitmq**
* **Mybatis**
* **Redis**
* **Spring Security**
* **Thymeleaf**
* **Markdown**
* **log4j**
* **elasticSearch**
* **alibaba fastjson**
* **alibaba Druid**
* **jQuery**
* **bootstrap**
* **autoMail**
* **google Thumbnailator**
* **swagger2**
